Pré-Requisitos

Antes de instalar qualquer coisa

Não pule essa etapa. Falta de espaço e falta de conta são os dois motivos mais comuns de tudo travar no meio do caminho.

  • 1 Verificar espaço livre: Mac: Ajustes do Sistema > Geral > Armazenamento (precisa de 15GB+). Windows: Configurações > Sistema > Armazenamento. Se estiver abaixo: apague vídeos antigos e esvazie a lixeira.
  • 2 Instalar WhatsApp Desktop: facilita copiar comandos para o Terminal sem erro de formatação. Mac: App Store. Windows: Microsoft Store. Procure "WhatsApp".
  • 3 Criar conta em claude.ai — use o mesmo email que vai usar no plano. Guarde o email e a senha.
  • 4 Assinar plano Pro em claude.ai — R$109/mês é o mínimo para começar. Sem o plano, o Claude Code não autentica.
Eduardo chegou com 6GB livres e o processo travou no meio. Faça a limpeza antes de começar — economiza tempo e dor de cabeça.

02

Instalar Node.js

O motor que faz o Claude Code funcionar

O Node.js é um programa obrigatório. Sem ele, o Claude Code simplesmente não instala. Instala uma vez, esquece para sempre.

  • Mac Acesse nodejs.org > clique em "macOS Installer" > Next > Next > Install > senha do Mac se pedir > FECHE o Terminal completamente > abra de novo > teste: node --version
  • Win Acesse nodejs.org > clique em "Windows Installer (LTS)" > Next > Install > FECHE o PowerShell completamente > reabra > teste: node --version
FECHAR e REABRIR o Terminal depois de instalar é obrigatório. Sem isso, o computador ainda não "conhece" o Node.js e vai aparecer "command not found".

03

Instalar Claude Code

A IA que roda no seu computador

Claude Code é diferente do site claude.ai. Ele roda no Terminal e executa tarefas reais no seu computador — não é só conversa.

  • Mac Abra o Terminal (Cmd + Espaço > digite "Terminal" > Enter). Digite: npm install -g @anthropic-ai/claude-code e aperte Enter. Se pedir senha: digite normalmente — os caracteres NÃO aparecem na tela, é normal! Aperte Enter ao terminar.
  • Win Abra o PowerShell (tecla Windows > digite "PowerShell" > Enter). Digite: npm install -g @anthropic-ai/claude-code
  • Err Se aparecer "permission denied": Mac: use sudo npm install -g @anthropic-ai/claude-code. Windows: feche e abra o PowerShell como Administrador (clique com botão direito > "Executar como administrador").
A senha do Mac nunca mostra os caracteres no Terminal. É um recurso de segurança. Pode digitar normalmente e apertar Enter — ele aceita.

Primeiro Acesso

Autenticar e entrar no Claude Code

A instalação terminou. Agora é hora de entrar pela primeira vez e autenticar com sua conta claude.ai.

  • 1 No Terminal, digite claude e aperte Enter
  • 2 Escolha o tema Escuro quando aparecer a opção
  • 3 O navegador vai abrir automaticamente para autenticar — faça login com o mesmo email do plano Pro
  • 4 Volte ao Terminal — se aparecer o prompt do Claude, está pronto!
Claude Code (Terminal) e claude.ai (site/app) são ferramentas diferentes. O Terminal executa tarefas. O site é para conversa. Use os dois de forma complementar.

Criar Estrutura de Pastas

Organizar antes de trabalhar

O Claude Code trabalha dentro de pastas. Criar a estrutura certa agora evita confusão depois — você sempre vai saber onde está cada projeto.

  • 1 Abra o Finder (Mac) ou Explorador de Arquivos (Windows), vá até Documentos, crie uma pasta chamada "Claude"
  • 2 Dentro de "Claude", crie duas subpastas: "Meu-Contexto" e "Otimizacao-PC"
  • Mac Clique com botão direito na pasta "Claude" > "Nova Aba de Terminal na Pasta" — o Terminal vai abrir já dentro da pasta certa
  • Win Abra a pasta "Claude" no Explorador > clique na barra de endereço no topo > apague o texto > digite powershell > Enter
  • 5 Com o Terminal aberto dentro da pasta, digite claude — agora você está dentro do projeto!
Sempre abra o Terminal de dentro da pasta do projeto. O Claude Code trabalha com os arquivos que estão na pasta onde você abre ele.
Troubleshooting

Travou? Calma.

Os erros mais comuns, traduzidos para humano, com a solução exata.

"command not found: claude"
O computador ainda não reconhece o comando. Solução: feche completamente o Terminal e abra de novo. Se persistir:
npm install -g @anthropic-ai/claude-code
"permission denied"
Sem permissão para instalar. Mac: adicione sudo antes do comando. Windows: feche e reabra o PowerShell como Administrador (botão direito).
sudo npm install -g @anthropic-ai/claude-code
"npm: command not found"
Node.js não está instalado — ou o Terminal ainda não foi fechado e reaberto após a instalação. Volte ao passo "Instalar Node.js" e repita o processo.
nodejs.org > instalar > FECHAR Terminal > abrir de novo
"Senha não aparece quando digito"
Isso é normal e esperado. O Mac esconde os caracteres da senha no Terminal por segurança. Continue digitando sua senha normalmente e aperte Enter.
Digite a senha + Enter — mesmo sem ver os caracteres
"ENOSPC: no space left"
Sem espaço em disco. Precisa de pelo menos 15GB livres. Apague vídeos antigos, esvazie a lixeira, e remova apps que não usa. Só depois tente de novo.
Mac: Ajustes > Geral > Armazenamento > libere 15GB
"Computador muito lento"
Claude Code precisa de pelo menos 8GB de RAM livres. Feche todos os outros programas antes de usar: navegador, Spotify, Zoom, tudo.
Feche outros programas > tente de novo
"Não consigo copiar do WhatsApp"
Copiar do WhatsApp no celular e colar no Terminal do Mac frequentemente dá erro de formatação. Solução: instale o WhatsApp Desktop no computador, ou mande o comando para si mesmo por email.
Instale WhatsApp Desktop > copie de lá
"Claude Code vs Claude app?"
Claude Code = Terminal — executa tarefas reais no seu computador, cria arquivos, roda comandos. Claude app/site = conversa — ideal para perguntas, rascunhos, ideias. São complementares, use os dois.
Terminal = executa. Site = conversa. Ambos são IA.

OMNIS na Prática

Quatro pilares para transformar sua empresa numa organização que aprende, lembra e evolui continuamente.

Pilar 01
Tudo se grava
Ative o OMNIS em TODA reunião. Sem exceção. As melhores ideias surgem quando ninguém está anotando — e é exatamente quando você mais precisa que tudo seja capturado.

Tudo tem dono
Depois de cada reunião, peça ao OMNIS: "Liste todas as tarefas mencionadas com responsável e prazo." Nunca mais uma decisão que fica no ar sem execução.

Treine sua equipe para consultar o OMNIS antes de decidir. Decisão sem contexto é chute.

Tudo evolui
Todo mês, peça um relatório de inteligência: tendências, padrões, oportunidades que ninguém percebeu. A empresa aprende como um organismo vivo — e você fica sempre um passo à frente.
Agende o "Relatório OMNIS Mensal" no calendário — primeiro dia de cada mês, sem falta.

Vocabulário

Tradutor para Leigos

Os termos que vao aparecer no seu caminho — traduzidos sem firula.

Terminal
Tela preta onde voce conversa com a IA e executa comandos. Mac: Cmd + Espaco > digite "Terminal". Windows: tecla Windows > "PowerShell".
Prompt
A instrucao que voce da para a IA. Quanto melhor a pergunta, melhor a resposta. Seja especifico, contextualize, diga o formato que quer.
Token
A moeda da IA. Cada palavra ou fragmento de texto consome tokens. Planos com mais tokens = mais capacidade de processar textos longos e complexos.
Skill
Habilidade que voce ensina ao Claude. Ele aprende na velocidade da luz. Uma skill pode ser "sempre escreva em tom direto" ou "use meu estilo de email".
Agente
Assistente com personalidade e autonomia definidas. Faz tarefas sozinho seguindo regras que voce estabelece. Diferente de uma conversa avulsa.
Squad
Time de 6 a 16 agentes trabalhando em conjunto. Como ter um departamento inteiro de IA rodando ao mesmo tempo numa so tarefa.
Node.js
Programa de computador necessario para o Claude Code funcionar. Voce instala uma vez e esquece. E como o motor embaixo do capo — nao precisa entender, so instalar.
Clone
Versao personalizada do Claude com suas skills, seu contexto e seu estilo. Quanto mais voce alimenta, mais ele parece voce — sem o cansaco.
OMNIS
Sistema da InvisIA que grava tudo na empresa — reunioes, decisoes, ideias. E a memoria de longo prazo da organizacao. Se nao gravou, nao aconteceu.
Sudo
Comando que da permissao de administrador no Terminal do Mac. E o "por favor com poderes maximos" do computador. Use quando aparecer "permission denied".
